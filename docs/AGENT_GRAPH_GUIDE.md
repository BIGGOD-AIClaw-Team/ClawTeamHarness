# Agent 图编辑指南

## 概述

Agent 图是 Agent 的核心，通过可视化拖拽的方式编排 Agent 的行为逻辑。

## 节点类型

| 节点类型 | 说明 | 用途 |
|---------|------|------|
| **Start** | 开始节点 | 入口点，每个图必须有且只有一个 |
| **LLM** | 大语言模型节点 | 调用 LLM 生成响应 |
| **Tool** | 工具节点 | 调用外部工具（如搜索、计算器） |
| **Condition** | 条件节点 | 根据条件分支 |
| **End** | 结束节点 | 出口点，标记流程结束 |

## 如何编辑

### 1. 创建 Agent
1. 点击「创建 Agent」按钮
2. 输入名称和描述
3. 点击「创建」

新创建的 Agent 会自带一个默认示例图（开始 → LLM 对话 → 结束），你可以在此基础上修改。

### 2. 编辑图
1. 在 Agent 列表中点击「编辑」按钮
2. 进入图编辑器

### 3. 添加节点
1. 点击「添加节点」按钮
2. 选择节点类型（Start / Node / End）
3. 节点会添加到图中

### 4. 连接节点
1. 节点四周有连接点（蓝色小圆点）
2. 拖拽一个节点的连接点到另一个节点
3. 建立数据流向

### 5. 保存图
1. 点击右上角「保存」按钮
2. 图会被保存到当前 Agent

### 6. 发布 Agent
1. 创建并保存图后
2. 点击「发布」按钮
3. 发布后即可在对话中使用

## 示例图

### 简单对话 Agent

```
[开始] → [LLM: 对话] → [结束]
```

这是一个最简单的 Agent，用户发送消息，LLM 直接回复。

### 带工具调用的 Agent

```
[开始] → [LLM: 理解意图] → [工具: 搜索] → [LLM: 组织回复] → [结束]
```

这个 Agent 会先理解用户意图，然后调用搜索工具，最后组织回复。

### 条件分支 Agent

```
[开始] → [LLM: 判断意图]
            |
    +-------+-------+
    |               |
   [搜索]        [对话]
    |               |
    +-------+-------+
            |
        [结束]
```

根据 LLM 判断，选择不同分支执行。

## 保存和发布

1. **保存**：只保存图，不影响线上版本（草稿状态）
2. **发布**：将当前图发布，用户才能在对话中使用
3. **取消发布**：撤回已发布的 Agent，恢复为草稿

## 注意事项

- 每个图必须有一个「开始」节点
- 每个图必须有一个「结束」节点
- 连接线表示数据流向
- 未发布的 Agent 不会出现在对话列表中
- 编辑前请先点击「编辑」进入图编辑器界面
